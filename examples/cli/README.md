# BrowseFleetMCP CLI Examples

These are direct command-line examples for running BrowseFleetMCP without going through an MCP client first.

- `basic.txt` starts the server directly.
- `custom-ports.txt` shows browser and broker port overrides.
- `auth-token.txt` shows both CLI and environment-variable auth token setup.

The same built server is used by the CLI and by MCP clients. The CLI just starts it directly.
