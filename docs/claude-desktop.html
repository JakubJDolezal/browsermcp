MCP Bundle

Package BrowseFleetMCP as a Claude Desktop extension.

Claude Desktop prefers a local MCP bundle for local tools. This repo now includes the root manifest.json and .mcpbignore needed to package the current checkout as an MCPB extension.

Bundle route

npm install -g @anthropic-ai/mcpb
npm run build
mcpb pack

Bundle files

What the repo already includes

  • manifest.json for Claude Desktop / MCPB metadata.
  • .mcpbignore to control bundle contents.
  • dist/index.js as the packaged Node entrypoint.

Install

How to load it in Claude Desktop

  1. Open Settings → Extensions.
  2. Open Advanced settings.
  3. Click Install Extension....
  4. Select the generated .mcpb file.

Behavior

What this route changes

Desktop-native install

Users install a packaged extension instead of editing config JSON by hand.

Same server core

The bundle still launches the same built Node server from dist/index.js.

Extension still required

The Chrome extension and connected-tab model do not go away just because Claude Desktop launches the MCP server differently.

Versioned package

This gives you a clean artifact to hand to Claude Desktop users or internal teams.
